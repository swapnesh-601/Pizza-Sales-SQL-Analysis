# 🍕 Pizza Sales Analysis using SQL

## 📌 Project Overview

This project focuses on analyzing Pizza Sales data using SQL. The objective is to derive meaningful business insights from sales transactions by performing data exploration, aggregation, joins, and advanced SQL analysis.

The project demonstrates practical SQL skills including data retrieval, filtering, grouping, aggregation, joins, subqueries, Common Table Expressions (CTEs), and revenue analysis.

---

## 🎯 Business Problem

A pizza restaurant wants to understand its sales performance, customer ordering behavior, and product popularity to make data-driven business decisions.

This project answers key business questions related to:

* Revenue generation
* Order patterns
* Pizza popularity
* Category performance
* Time-based sales trends
* Revenue contribution analysis

---

## 🗂️ Dataset Information

The project uses four relational datasets:

### orders.csv

Contains order information including order ID and order date/time.

### order_details.csv

Contains details of pizzas included in each order.

### pizzas.csv

Contains pizza sizes and prices.

### pizza_types.csv

Contains pizza names, categories, and ingredients.

## 📊 Analysis Performed

### Basic Analysis

* Retrieve the total number of orders placed.
* Calculate total revenue generated.
* Identify the highest-priced pizza.
* Find the most common pizza size ordered.
* Determine the top 5 most ordered pizza types.

### Intermediate Analysis

* Calculate quantity ordered by pizza category.
* Analyze order distribution by hour.
* Determine category-wise pizza distribution.
* Calculate average pizzas ordered per day.
* Identify top 3 pizza types by revenue.

### Advanced Analysis

* Calculate percentage contribution of each pizza type to total revenue.
* Analyze cumulative revenue over time.
* Determine top 3 revenue-generating pizzas within each category.

## 🚀 How to Run the Project

### Step 1: Create Database

```sql

```

### Step 2: Import CSV Files

Import the following files into your SQL database:

* orders.csv
* order_details.csv
* pizzas.csv
* pizza_types.csv

### Step 3: Execute SQL Script

Run:

```sql
Pizza_sales.sql
```

to perform all analyses and generate insights.
